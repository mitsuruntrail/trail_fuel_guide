# trail_fuel_guide
トレイル補給ナビ（Trail Fuel Guide）

トレイルランニング向けの補給計算・アドバイスアプリです。
距離・累積標高・行動時間・走り方から、
必要な水分量・補給食・電解質（OS-1など）の目安を算出します。

現在は β版 として公開しています。

⸻

できること
	•	行動時間と標高差を考慮した水分量の目安を計算
	•	補給食（ジェル・固形物・電解質）の必要数を算出
	•	初心者にも分かりやすい補給アドバイスを表示
	•	長時間トレイルを想定した安全側の設計

⸻

想定ユーザー
	•	トレイルランニング初心者〜中級者
	•	初めて長めの距離・累積標高に挑戦する人
	•	「何をどれくらい持てばいいか分からない」人

※ エリート・トップ選手向けのピーキーな設計ではありません。

⸻

使い方（簡易）
	1.	距離・累積標高・想定行動時間を入力
	2.	走り方（速い / 普通 / ゆっくり）を選択
	3.	「計算する」を押す
	4.	補給プランと推奨アイテムを確認

⸻

注意事項（必ずお読みください）
	•	本アプリの内容は医療行為・治療・診断を目的としたものではありません
	•	補給量はあくまで目安です
	•	気温・体質・発汗量・当日の体調によって調整してください
	•	実際のレースや山行では事前にテスト（練習）することを強く推奨します
